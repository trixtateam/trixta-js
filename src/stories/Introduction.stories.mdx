# Welcome to Storybook for Trixta

The sidebar is split into four sections COMMON, TRIXTA ACTIONS , TRIXTA REACTIONS and TRIXTA REDUX STORE.

<div className="tip-wrapper">
  <span className="tip">Select a Trixta role</span> above in the toolbar, this will be used for all future
  Trixta storybook stories that require the <code>roleName</code> parameter
</div>

## Common Section
This section has all functionality of Trixta that does not require a action or reaction name.

## Trixta Actions Section
This section has all functionality of Trixta that does require a action name.
<div className="tip-wrapper">
  <span className="tip">Select a Trixta action</span> above in the toolbar, this will be used for all future
  Trixta storybook stories that require the <code>actionName</code> parameter
</div>

## Trixta Reactions Section
This section has all functionality of Trixta that does require a reaction name.
<div className="tip-wrapper">
  <span className="tip">Select a Trixta reaction</span> above in the toolbar, this will be used for all future
  Trixta storybook stories that require the <code>reactionName</code> parameter
</div>
